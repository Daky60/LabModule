# LabModule
A Powershell module for automatic deployment of test environments.   
You can create virtual switches, VMs and forests and join said VMs to it, among with other things.  
Available functions:  
Build-LabSwitch  
Build-LabVM  
Build-LabForest  
Remove-LabVM  
Remove-LabEnv
